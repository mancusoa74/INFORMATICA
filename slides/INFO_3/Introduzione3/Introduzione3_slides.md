# Informatica

Una definizione

<div style="background-color: green; color: yellow; padding: 2rem; font-size: 2rem; line-height: 3rem;">
L'informatica è la scienza che si occupa del trattamento dell'informazione mediante procedure automatizzate, avendo per oggetto lo studio dei fondamenti teorici dell'informazione, della sua computazione a livello logico e delle tecniche pratiche per la sua implementazione.
</div>

# Il Perito Informatico

Una definizione

<div style="background-color: green; color: yellow; padding: 1rem; font-size: 2rem; line-height: 3rem;">
<p style="line-height: 3rem;">Il perito informatico è una figura professionale prevista dal sistema scolastico italiano che opera nel campo dell'informatica.</p>

# Il Perito Informatico

Una definizione

<div style="background-color: green; color: yellow; padding: 1rem; font-size: 2rem; line-height: 3rem;">

<p style="line-height: 3rem;">

Inoltre ha le conoscenze di base necessarie per partecipare alla realizzazione e alla gestione di grandi sistemi di automazione basati sull'elaborazione dell'informazione digitale anche online.
</p>
